## ðŸš€ Quick Start Guide

### Step 1: Import All Dashboards (10 minutes)

```bash
# SSH to Grafana server (mentat)
ssh root@51.254.139.78

# Clone repository
cd /tmp
git clone https://github.com/calounx/mentat.git

# Copy all dashboards to Grafana
sudo mkdir -p /var/lib/grafana/dashboards/{advanced,business,performance,cost,security,data}

sudo cp /tmp/mentat/deploy/grafana-dashboards/advanced/*.json \
   /var/lib/grafana/dashboards/advanced/

sudo cp /tmp/mentat/deploy/grafana-dashboards/business/*.json \
   /var/lib/grafana/dashboards/business/

sudo cp /tmp/mentat/deploy/grafana-dashboards/performance/*.json \
   /var/lib/grafana/dashboards/performance/

sudo cp /tmp/mentat/deploy/grafana-dashboards/cost/*.json \
   /var/lib/grafana/dashboards/cost/

sudo cp /tmp/mentat/deploy/grafana-dashboards/security/*.json \
   /var/lib/grafana/dashboards/security/

sudo cp /tmp/mentat/deploy/grafana-dashboards/data/*.json \
   /var/lib/grafana/dashboards/data/

# Set permissions
sudo chown -R grafana:grafana /var/lib/grafana/dashboards/
sudo systemctl restart grafana-server

# Verify
curl http://localhost:3000/api/search?type=dash-db | jq '. | length'
# Should show: 22 (5 original + 17 new)
```

### Step 2: Configure Data Sources (5 minutes)

All dashboards use **Prometheus** as the data source.

```bash
# Verify Prometheus is configured
curl http://localhost:3000/api/datasources | jq '.[] | select(.type=="prometheus")'

# If not configured, add it
curl -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <GRAFANA_API_KEY>" \
  -d '{
    "name": "Prometheus",
    "type": "prometheus",
    "url": "http://localhost:9090",
    "access": "proxy",
    "isDefault": true
  }' \
  http://localhost:3000/api/datasources
```

### Step 3: Install Required Exporters (10 minutes)

```bash
# Already installed:
# - Node Exporter (system metrics)
# - MySQL Exporter (database metrics)

# Install additional exporters as needed:

# PHP-FPM Exporter (for APM dashboard)
wget https://github.com/hipages/php-fpm_exporter/releases/download/v2.2.0/php-fpm_exporter_2.2.0_linux_amd64
sudo mv php-fpm_exporter_2.2.0_linux_amd64 /usr/local/bin/php-fpm_exporter
sudo chmod +x /usr/local/bin/php-fpm_exporter

# Create systemd service
sudo tee /etc/systemd/system/php-fpm-exporter.service > /dev/null <<EOF
[Unit]
Description=PHP-FPM Exporter
After=network.target

[Service]
Type=simple
ExecStart=/usr/local/bin/php-fpm_exporter --phpfpm.scrape-uri=tcp://127.0.0.1:9000/status

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl daemon-reload
sudo systemctl enable php-fpm-exporter
sudo systemctl start php-fpm-exporter
```

### Step 4: Implement Application Metrics (30 minutes)

Use the provided implementation guides:

**For Laravel/PHP (CHOM):**
- `/deploy/grafana-dashboards/cost/MetricsExporter.php.example`
- `/deploy/grafana-dashboards/business/IMPLEMENTATION_GUIDE.md`

**For Python applications:**
- `/deploy/grafana-dashboards/advanced/example-metrics.py`

### Step 5: Configure Alerts (15 minutes)

```bash
# Add Prometheus alert rules
sudo cp /tmp/mentat/deploy/grafana-dashboards/*/alert-rules.yml \
   /etc/prometheus/rules/

# Verify syntax
promtool check rules /etc/prometheus/rules/*.yml

# Reload Prometheus
sudo systemctl reload prometheus

# Configure Alertmanager (already done)
# Alerts will be sent via Brevo email
```
