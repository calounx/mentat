## Already Experienced? Fast Track

If you know hosting and want to get straight to business:

**For deployment:**
```bash
# 1. Clone repo (30 seconds)
git clone https://github.com/calounx/mentat.git
cd mentat/chom/deploy

# 2. Configure (5 minutes)
cp configs/inventory.yaml.example configs/inventory.yaml
nano configs/inventory.yaml  # Add your VPS IPs

# 3. Deploy (20 minutes)
./deploy-enhanced.sh --validate  # Check setup
./deploy-enhanced.sh all         # Deploy everything
```

**For local development:**
```bash
# 1. Clone and install (5 minutes)
git clone https://github.com/calounx/mentat.git
cd mentat/chom
composer install && npm install

# 2. Configure (2 minutes)
cp .env.example .env
php artisan key:generate
touch database/database.sqlite

# 3. Run (1 minute)
php artisan migrate
composer run dev  # Starts server + watch
```

Done! See [Getting Started](/home/calounx/repositories/mentat/chom/docs/GETTING-STARTED.md) for details.
